## Description

```
Privilege is a local-first engagement confidentiality preflight for consultants.

Redaction hides names. A frontier model can still re-identify a client from context — the mosaic. Privilege masks declared terms on your laptop, turns GPT-5.6 into a blind attacker against the sanitized draft, then lets you download an anonymized PDF for ChatGPT (or any AI) and restore real names locally from the reply.

Demo loop:
• Set engagement policy (Northwind Freight / Baltic corridor)
• Upload & attest client PDF
• Anonymize → Transform (mosaic found) → download anonymized PDF
• Attach that PDF in ChatGPT
• Paste reply → restore names on-device

Built for OpenAI Build Week (Work & Productivity).
Core built in Codex with GPT-5.6; PDF/attestation UI finished after the Codex quota (disclosed in the README).
```
